# 2nite


## Controls
- Arrows, W, A, D - Movement
- Q - Build wall (Hold for placement visual)
- E - Build platform (Hold for placement visual)
- F - Build floor (Hold for placement visual)
- R - Reload
- 1 - SMG
- 2 - Rifle
- 3 - Shotgun
- 4 - Sniper
- Spacebar - Jump
- Mouse - Aim
- Left-click - Shoot (Hold for auto fire)
